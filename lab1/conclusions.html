Висновки

У результаті виконання лабораторної роботи було набуто практичних навичок роботи з GitHub та репозиторіями, що дозволяє організовано зберігати і керувати проектами. Також було опрацьовано основи створення HTML-документів та структурування інформації за допомогою таблиць, зображень, посилань, списків і форм. Створений шаблон звітного HTML-документа може бути використаний для представлення результатів виконання всіх лабораторних робіт.

Застосування вивчених елементів сприяло розумінню базових принципів веб-розробки, формуванню вміння створювати інформативні та зручні у використанні веб-сторінки. Отримані знання та навички є фундаментом для подальшого вивчення сучасних технологій веб-програмування.
