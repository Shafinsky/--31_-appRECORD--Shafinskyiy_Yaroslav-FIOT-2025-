Способи додавання CSS стилів на сторінку

1. Зовнішній файл стилів

Цей метод дозволяє додавати стилі безпосередньо до HTML-елементів. Кожен елемент стилізується окремо, що робить цей підхід зручним для локальних або одноразових змін.

Приклад:

<link rel="stylesheet" href="style.css">

Переваги:

  • Легкість у використанні для окремих елементів.

Недоліки:

  • Погано підходить для великих проєктів.
  • Важко підтримувати та змінювати стилі в майбутньому.
  • Може призвести до дублювання стилів.
