Cелектор класу

Використано для стилізації блоків інтерфейсу (хедер, меню, кнопки тощо):

HTML-код та CSS-код

        .header {
  display: flex;
  align-items: center;
  justify-content: space-between;
  background: #111;
  padding: 10px 40px;
}

.logo-area {
  display: flex;
  align-items: center;
  gap: 10px;
}

.sign-in {
  background: #00ff66;
  color: #000;
  padding: 5px 12px;
  border-radius: 4px;
}

Зображення

Теорія

Селектор класу — це тип CSS-селектора, який вибирає елементи HTML, які мають певний клас. Клас визначається в атрибуті class у HTML-елементі. Селектори класів починаються з крапки (.) перед ім'ям класу.

Наприклад, якщо ви хочете стилізувати всі елементи з класом highlight, ви можете використовувати селектор класу так:


        .highlight {
            background-color: yellow; /* Змінює фон елементів з класом highlight на жовтий */
            font-weight: bold; /* Робить текст жирним */
        }

У HTML це виглядатиме так:


        <p class="highlight">Цей текст виділений жовтим фоном.</p>
        <p>Цей текст не має спеціального стилю.</p>

В цьому прикладі перший абзац отримає жовтий фон і жирний текст, тоді як другий залишиться без змін.
