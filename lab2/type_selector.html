Cелектор тегу

Застосовані для основних елементів сторінки:

HTML-код та CSS-код (на прикладі <form>)

        body {
  font-family: "Segoe UI", Arial, sans-serif;
  background: #1a1a1d;
  color: #fff;
}

h2 {
  color: #00ff66;
  margin-top: 30px;
}

table {
  width: 100%;
  border-collapse: collapse;
}

Зображення

Теорія

Селектор тегу — це базовий тип CSS-селектора, який вибирає всі елементи HTML з певним тегом. Наприклад, якщо необхідно стилізувати всі абзаци (<p>) на веб-сторінці, використовуємо селектор тегу p. Ось приклад:


            p {
            color: blue; /* Змінює колір тексту всіх абзаців на синій */
            font-size: 16px; /* Встановлює розмір шрифту на 16 пікселів */
            }

У цьому прикладі всі абзаци в документі матимуть синій текст і розмір шрифту 16 пікселів.
